## 🔄 دورة حياة الجلسة:

```
1. الإنشاء
   └─► generateSessionId()
       └─► session_abc123_1707234567

2. عرض الـ Modal
   └─► showSessionModal()
       ├─► عرض الـ Session ID
       ├─► عرض الـ Link
       └─► عرض الأزرار

3. المشاركة
   └─► خيارين:
       ├─► copyToClipboard()
       │   └─► نسخ الرابط
       └─► shareViaWhatsApp()
           └─► فتح WhatsApp

4. الانضمام
   └─► صديقك يفتح الرابط
       └─► joinStudySession()
           └─► أضفنا في الـ members

5. التخزين
   └─► localStorage.studySessions
       └─► حفظ كل البيانات
```

## 🚨 حالات الخطأ:

```
الحالات المتوقعة:

1️⃣ لم يسجل دخول؟
   └─► يروح على صفحة login تلقائياً

2️⃣ اضغط خارج الـ Modal؟
   └─► ينغلق Modal وترجع لصفحتك

3️⃣ الرابط ما اتنسخ؟
```
